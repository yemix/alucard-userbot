# alucard-userbot
This is a bot logged in as a user. The code was used from alucard-selfbot!
- play music
- selfbot commands

### Install
Open command promp and run "python3 -m pip install -r requirements.txt.

After that enter your login-details in config.json.

Launch launch.bat.
